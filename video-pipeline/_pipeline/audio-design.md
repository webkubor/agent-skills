# 声音设计 Skill

> **管线位置**: 专项技能模块 — 为 Seedance prompt 提供音频层设计规范
> **注意**: Seedance 2.0 支持原生音频生成（音效+音乐+台词同步）

---

## 一、音频层次

每个场景的声音由四层叠加：

| 层次 | 内容 | 优先级 | 典型音量 |
|------|------|--------|----------|
| 1. 台词 | 人物对话 | 最高 | 主音量 |
| 2. 动作音效 | 同步于画面动作 | 高 | 稍低于台词 |
| 3. 环境音 | 场景背景声 | 中 | 铺底 |
| 4. 音乐 | 情绪/节奏指引 | 中低 | 情绪烘托 |

## 四、台词设计规范

### Seedance 台词编写
```text
台词1（[角色标签]，[情绪状态]）："[台词内容]"
```

### 情绪标注词汇
```
愤怒、悲伤、喜悦、冷漠、嘲讽、温柔、恐惧、坚定、
疲惫、惊讶、哽咽、低声、嘶吼、颤抖、平静
```

### 台词节奏控制
- 短台词（3-5字）：用于关键时刻，一字一顿
- 中等台词（8-15字）：标准对话节奏
- 长台词（15+字）：用于独白/内心戏，需配合运镜变化

## 五、音乐情绪映射

| 场景情绪 | 音乐风格建议 |
|----------|-------------|
| 紧张/悬疑 | 低频弦乐、渐强鼓点、不和谐音程 |
| 温情/感动 | 钢琴独奏、弦乐柔板、吉他指弹 |
| 燃/热血 | 强节奏电子、交响乐齐奏、鼓点密集 |
| 悲伤/失落 | 大提琴独奏、慢板钢琴、环境留白 |
| 武侠/古风 | 古筝、箫、琵琶、大鼓 |
| 都市/现代 | 电子、Lo-fi、爵士 |
| 恐怖/惊悚 | 噪音、尖锐弦乐、不规则节奏 |

## 六、音频同步提示

### 画面-音效同步
```text
[画面动作] + 伴随 [音效] + 声画同步
```
示例: `剑刃劈下，伴随金属碰撞声，火花四溅`

### 台词-口型注意
- Seedance 2.0 口型同步尚不稳定，
  建议台词对应的镜头用近景/特写（面部占比较大时口型更容易匹配）
- 长台词考虑拆分为短句，分布在多个镜头

---

## 七、声音设计检查清单

- [ ] 每层音频是否都覆盖到了
- [ ] 环境音是否与场景匹配
- [ ] 动作音效是否与关键动作同步
- [ ] 台词标注了角色标签和情绪状态
- [ ] 音乐风格是否匹配情绪曲线
- [ ] 各层音量是否有主次区分
