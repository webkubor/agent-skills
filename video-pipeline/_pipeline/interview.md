# 需求访谈 Skill

> **管线位置**: 第 1 站 — 将模糊想法转化为结构化的创意简报
> **输入**: 用户自然语言描述
> **输出**: 创意简报（Creative Brief）
> **参考**: Emily2040/seedance-2.0 的 seedance-interview 模块

---

## 目标

通过 5-8 个关键问题，将用户的模糊创意转化为包含所有必要参数的创意简报。

---

## 访谈流程

### 第一步：打开话题

用户说「我想做一个XX视频」→ 然后问的第一个问题：

> 「好的，我帮你把这个创意落地。先确认几个关键信息——」

### 第二步：核心问题（按优先级排列）

1. **主题与情绪**: 「这个视频想讲什么故事？希望观众看完有什么感觉？」
   - 记录：一句话主题概括 + 目标情绪

2. **目标平台**: 「发到哪里？抖音/小红书/视频号/B站？」
   - 确定：画幅比例（9:16 竖屏 / 16:9 横屏 / 1:1 方形）

3. **时长**: 「大概多长？」
   - 确定：单段（4-15秒）/ 多段拼接（>15秒）

4. **参考素材**: 「有没有现成的图片或视频可以参考？」
   - 角色参考图 / 场景参考图 / 首帧图 / 风格参考视频 / 音频素材

5. **风格偏好**: 「画面风格上有什么偏好吗？」
   - 写实 / 电影感 / 动漫 / 插画 / 写意
   - 色调偏好 / 节奏偏好

6. **关键约束**: 「有什么绝对不能出现的？」
   - 禁忌元素 / 平台限制 / 品牌规范

### 第三步：确认简报

将以上信息整理为创意简报，回传给用户确认：

---

## 创意简报模板

```markdown
## 创意简报

### 基本信息
- **项目名称**: [名称]
- **目标平台**: [平台名]
- **画幅**: [16:9 / 9:16 / 1:1]
- **总时长**: [X] 秒
- **参考素材**: [清单]

### 创意核心
- **一句话主题**: [主题概括]
- **目标情绪**: [1-3个情绪关键词]
- **核心视觉元素**: [必须出现的元素]

### 风格偏好
- **视觉风格**: [风格类型]
- **色调方向**: [色调描述]
- **节奏偏好**: [快/中/慢]

### 禁忌清单
- ❌ [不可出现的元素1]
- ❌ [不可出现的元素2]

### 特殊需求
- [任何额外需求]
```

---

## 访谈技巧

- **不要一次性问所有问题**: 先问最关键的 2-3 个，根据回答引导后续问题
- **提供选项而非开放问题**: 「偏暖调还是冷调？」比「什么色调？」更好回答
- **用户不知道的不强求**: 如果用户说「我也不确定」，给一个合理默认值
- **用示例引导**: 「像《东邪西毒》那种感觉？」比「什么风格？」更有效
