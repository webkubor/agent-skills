# 镜头语言库

> **管线位置**: 专项技能模块 — 为 prompt-compiler 提供镜头运镜的标准词汇
> **参考**: 专业电影摄影术语

---

## 一、景别（Shot Size）

| 景别 | 英文关键词 | 构图重点 | 适用场景 |
|------|-----------|----------|----------|
| 大远景 | extreme long shot, establishing shot | 环境为主，人物极小 | 世界观建立、转场 |
| 全景 | full shot, wide shot | 人物全身 + 环境 | 动作戏、走位展示 |
| 中全景 | medium full shot | 人物膝盖以上 | 多人对话 |
| 中景 | medium shot, mid shot | 人物腰部以上 | 标准对话、动作中景 |
| 中近景 | medium close-up | 人物胸部以上 | 情感对话、反应 |
| 近景 | close-up | 人物面部 | 情绪传达、关键台词 |
| 特写 | extreme close-up | 单一眼/手/物品 | 细节强调、情绪放大 |
| 插入镜头 | insert shot | 与主情节相关的物品 | 线索提示、伏笔 |

## 二、运镜（Camera Movement）

### 基础运镜
| 运镜 | 英文关键词 | 视觉效果 |
|------|-----------|----------|
| 推近 | push in, dolly in | 逐渐接近主体，增强紧张感 |
| 拉远 | pull out, dolly out | 逐渐远离，揭示环境或产生孤独感 |
| 平移 | pan left/right | 水平扫视，跟随运动或展示场景 |
| 俯仰 | tilt up/down | 垂直扫视，揭示高度或展示层次 |
| 跟踪 | tracking shot, follow | 跟随主体运动，增强代入感 |
| 环绕 | orbit, arc shot | 围绕主体旋转，展示立体感 |
| 升降 | crane up/down, pedestal | 垂直运动，改变观众高度视角 |
| 变焦 | zoom in/out | 改变焦距放大/缩小画面 |

### 进阶运镜
| 运镜 | 英文关键词 | 视觉效果 |
|------|-----------|----------|
| 推拉变焦 | dolly zoom, vertigo shot | 主体不变背景变形，表现心理震撼 |
| 手持 | handheld, shaky cam | 纪录感、真实感、紧张感 |
| 斯坦尼康 | steadicam, gimbal | 平滑稳定的运动跟踪 |
| 航拍 | drone shot, aerial, bird's eye | 高空俯瞰，宏大叙事 |
| 滑轨 | slider, dolly track | 平滑线性运动 |
| 摇臂 | jib, crane | 大范围垂直/弧线运动 |
| 快速摇镜 | whip pan, swish pan | 快速甩镜，用于转场或紧迫感 |
| 慢镜头 | slow motion | 时间拉长，强调动作美感 |
| 快镜头 | speed ramp, fast motion | 时间压缩，表现过程或喜剧效果 |

## 三、角度（Camera Angle）

| 角度 | 英文关键词 | 心理效果 |
|------|-----------|----------|
| 平视 | eye level | 平等、自然 |
| 仰拍 | low angle | 强大、权威、压迫感 |
| 俯拍 | high angle, overhead | 脆弱、渺小、被审视 |
| 荷兰角 | dutch angle, canted | 不安、失衡、混乱 |
| POV | point of view, POV shot | 第一人称代入 |

---

## 四、焦距与景深

| 效果 | 英文关键词 | 技术说明 |
|------|-----------|----------|
| 浅景深 | shallow depth of field, f/1.4 - f/2.8 | 主体清晰背景虚化 |
| 深景深 | deep focus, f/8 - f/16 | 前景到远景都清晰 |
| 广角 | wide angle, 24mm, 35mm | 视野宽广，增强空间感 |
| 标准 | standard, 50mm | 接近人眼视角 |
| 长焦 | telephoto, 85mm, 135mm, 200mm | 压缩空间，突出主体 |

## 五、常见的镜头组合套路

### 引入式（3镜头）
```
1. 大远景 — 建立场景
2. 中景 — 引入人物
3. 近景 — 聚焦主体
```

### 对话正反打（3镜头）
```
1. 中景双人镜头 — 建立关系
2. 近景 A — A 说话
3. 近景 B — B 反应
```

### 动作分解（3镜头）
```
1. 中景 — 动作起势
2. 特写 — 关键瞬间
3. 全景 — 动作落点/后果
```

### 悬念揭示（3镜头）
```
1. 近景 — 人物表情反应
2. 缓慢推近 — 引导观众注意
3. 特写 — 揭示关键信息
```
