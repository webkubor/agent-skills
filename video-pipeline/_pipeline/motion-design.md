# 动作设计 Skill

> **管线位置**: 专项技能模块 — 为 Seedance prompt 提供动作描述的标准词汇
> **关联**: 与 `video/_pipeline/camera-language.md` 和 `video/cinematic-action/SKILL.md` 配合使用

---

## 一、动作描述公式

```
[肢体部位] + [动作动词] + [幅度/速度/力度]
```

### 示例
| 基础描述 | 精确描述 |
|----------|----------|
| 抬手 | 右手缓慢抬至胸前，五指微张 |
| 转头 | 头部快速向左转，发丝随之甩动 |
| 走路 | 步伐缓慢而沉稳，每一步踏地有力 |
| 微笑 | 嘴角微微上扬，笑意未达眼底 |
| 挥剑 | 单手横挥长剑，剑刃划出一道弧光 |

## 三、动作节奏模型

### 三段式动作弧线

```
起势（蓄力）→ 过程（释放）→ 收势（余韵）
2-3秒          3-5秒          2-3秒
```

### 节奏类型与应用

| 节奏 | 特征 | 适用场景 | 镜头时长建议 |
|------|------|----------|-------------|
| 快速爆发 | 蓄力短→释放快→余韵长 | 战斗、突袭、冲突 | 0.5-1.5s / 镜头 |
| 中速推进 | 三者均衡 | 追逐、对话推进 | 2-3s / 镜头 |
| 缓慢蓄势 | 蓄力长→释放短→余韵长 | 对峙、内心戏 | 4-6s / 镜头 |
| 静中有动 | 主体静止，环境/细节运动 | 氛围、情绪留白 | 5-8s / 镜头 |

## 四、特殊动作类型

### 武侠/打斗
```yaml
兵器出鞘: 拇指推剑格，剑身缓缓滑出剑鞘，金属摩擦声
剑气: 横挥一剑，空气被撕裂，远处树叶被无形之力震落
轻功: 脚尖轻点竹叶，身体腾空向上，衣袍猎猎作响
内力: 双掌缓缓推出，地面砂石被气流卷起，缓慢旋转
```

### 舞蹈/身体表现
```yaml
旋转: 身体缓慢旋转，裙摆如花开般展开，头发随之飘散
跳跃: 屈膝蓄力，猛力蹬地跃起，在空中定格一瞬
流畅: 动作如水般连绵不断，一个动作自然过渡到下一个
```

### 日常/自然动作
```yaml
行走: 步伐节奏均匀，手臂自然摆动，衣料轻微摩擦
喝水: 缓缓举杯，嘴唇轻触杯沿，喉结微微滚动
书写: 毛笔在宣纸上划过，墨迹慢慢晕开
```

## 五、动作与运镜的配合

| 动作类型 | 推荐运镜 | 原因 |
|----------|----------|------|
| 快速运动（奔跑/打斗） | 跟踪镜头 / 手持 | 保持主体在画内，增强代入感 |
| 缓慢动作（沉思/行走） | 固定机位 / 缓慢推近 | 让观众安静观察 |
| 旋转动作 | 环绕镜头 | 展示主体的立体动态 |
| 跳跃/上升 | 仰拍 / 升降镜头 | 增强高度感和力量感 |
| 降落/下落 | 俯拍 / 镜头跟随下降 | 展示垂直空间关系 |
| 对峙/静止 | 缓慢推近 + 反应镜头 | 积累紧张感 |

## 六、Prompt 写法示例

### ❌ 不好的动作描述
```
人物在打斗，很精彩
```

### ✅ 好的动作描述
```
[主体] 右手缓慢拔剑出鞘，剑刃与鞘口摩擦出细微火花，
左手自然下垂，身体微微侧转，眼神冷峻锁定前方，
剑尖缓缓抬起指向对手，呼吸平稳，衣袍被微风轻拂
```
