# 分镜脚本 Skill

> **管线位置**: 第 2 站 — 从创意简报到镜头级分镜脚本
> **输入**: 创意简报 + 风格目录（可选）
> **输出**: 镜头级分镜脚本（故事板）

---

## 目标

将创意简报转化为精确到镜头级别的分镜脚本，每个镜头包含画面描述、运镜、台词、音效、时长。

---

## 分镜设计原则

1. **一个镜头 = 一个视觉意图**: 每个镜头只完成一件事（引入 / 推进 / 转折 / 收束）
2. **因果推进**: 镜头 N 的画面是镜头 N+1 的前提
3. **视觉层次**: 景别从全景 → 中景 → 特写 → 全景收束
4. **节奏控制**: 3-5 秒一个镜头，慢情绪可到 8 秒

---

## 景别选择指南

| 景别 | 英文 | 用途 | 典型时长 |
|------|------|------|----------|
| 大远景 | extreme long shot | 建立场景/世界观 | 3-5s |
| 全景 | full shot | 人物与环境关系 | 3-4s |
| 中景 | medium shot | 人物动作/对话 | 3-4s |
| 近景 | close-up | 人物表情 | 2-3s |
| 特写 | extreme close-up | 关键细节/情绪放大 | 1-2s |

---

## 分镜脚本模板

```markdown
## 分镜脚本

### 项目信息
- **标题**: [项目名称]
- **总时长**: [X] 秒
- **分镜数**: [N] 个镜头
- **画幅**: [16:9 / 9:16 / 1:1]

---

### 镜头 1（0-[X]秒）— [镜头功能]
- **景别**: [大远景 / 全景 / 中景 / 近景 / 特写]
- **运镜**: [推近 / 拉远 / 跟踪 / 环绕 / 固定 / 手持 / 摇镜]
- **画面**: [详细描述画面内容：人物、场景、动作]
- **台词**: [角色名]："[台词内容]"（如无则写"无"）
- **音效**: [环境音 / 动作音 / 留白]
- **衔接**: 切入镜头 2 — [过渡方式：硬切 / 叠化 / 匹配剪辑]

### 镜头 2（[X]-[Y]秒）— [镜头功能]
...（重复上述结构）

### 镜头 N（[Y]-[Z]秒）— 收束镜头
- **景别**: ...
- **运镜**: ...
- **画面**: 定格在 [描述收束画面]
- **台词**: ...
- **音效**: 渐弱收束
```

## 特殊场景处理

### 对话戏
- 建立镜头（二人位置关系）→ 正反打（谁说话给谁镜头）→ 反应镜头（听者表情）
- 台词与画面分离：一句话可以拆分到两个镜头中

### 动作戏
- 动作分解：一个连贯动作拆成 2-3 个镜头（起势→过程→落点）
- 节奏变化：快动作用短镜头（1-2s），蓄力用慢镜头（3-4s）

### 产品展示
- 建立产品 → 细节特写 → 使用场景 → Logo 定格
- 运镜：缓慢推近 + 360 度旋转 + 光效过渡

### 社交媒体短片
- 前 2 秒必须有视觉钩子（最惊艳的画面放最前面）
- 每个镜头 ≤3 秒
- 结尾必须有 call-to-action 画面

---

## 质量检查

- [ ] 镜头功能是否可识别（每个镜头有明确的叙事目的）
- [ ] 因果关系是否成立（观众能理解为什么从镜头 N 到 N+1）
- [ ] 节奏是否合理（情绪紧张时短镜头，舒缓时长镜头）
- [ ] 视觉层次是否丰富（景别有明显变化，不全是一个景别）
- [ ] 总时长是否匹配目标
