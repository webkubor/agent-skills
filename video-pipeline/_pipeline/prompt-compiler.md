# 提示词编译器 Skill

> **管线位置**: 第 3 站 — 从分镜脚本到可直接使用的 Seedance 2.0 prompt
> **输入**: 分镜脚本 + 参考素材清单
> **输出**: 生产级 Seedance 2.0 提示词（单段或多段拼接）

---

## 目标

将分镜脚本中的自然语言镜头描述，编译为 Seedance 2.0 可直接使用的结构化提示词。

---

## 编译规则

### 基础公式
```
Seedance Prompt = 主体定义 + 分镜时序 + 动作描述 + 运镜指令 + 风格锚定 + 画质参数 + 负面约束
```

### 主体定义规则
- 有参考图 → 先定义主体：`将 <图片N> 中的 [2-3个特征] 定义为 <主体1>`
- 无参考图 → 在第一个镜头中详细描述主体特征
- 多角色 → 每个角色单独定义，使用稳定标签

### 分镜编译规则
- 将分镜脚本的「镜头 1 / 镜头 2 / 镜头 3」结构保留
- 每个镜头编译为：
  ```
  [时间区间]：[运镜方式] + [主体标签] + [动作描述] + [场景环境] + [台词] + [音效]
  ```
- 去掉景别标注（景别由运镜和构图的英文关键词表达）

### 语言要求
- 主体和中文台词保持中文
- 运镜、风格、画质关键词使用英文（参见 `models/seedance-2.0.md` 运镜速查）
- 负面约束使用英文

---

## 编译模板

### 单段视频（≤15秒）

```text
[总时长]秒 [主题概括]，[分镜时序]：

镜头1（0-[X]秒）：[运镜(en)] [主体标签] [动作(zh)] [场景(zh)] [台词(zh)] [音效(zh)]
镜头2（[X]-[Y]秒）：[运镜(en)] [主体标签] [动作(zh)] [场景(zh)] [台词(zh)] [音效(zh)]
镜头3（[Y]-[Z]秒）：[运镜(en)] [主体标签] [收束动作(zh)] [音效渐弱]

[风格锚定]: cinematic, [具体风格词], film grain, desaturated tones, 
natural lighting, 35mm lens aesthetic

[画质参数]: 2K resolution, smooth motion, stable composition

[负面约束]: no shaky camera, no deformation, no extra limbs, 
no plastic skin, no random text, no watermark, no subtitles
```

### 多段拼接（>15秒）

每段单独输出，标注衔接点和延长操作。参见 `models/seedance-2.0.md` 第三章多段拼接模板。

---

## 输出要求

1. **输出 2-3 个版本**（A/B/C），核心差异在运镜和节奏上
2. **版本 A**: 社交封面冲击力（快节奏、视觉钩子前置）
3. **版本 B**: 电影氛围故事张力（慢节奏、情绪铺垫）
4. **版本 C**: 人物气质情绪留白（聚焦人物、减少场景信息）

---

## 参考素材引用规则

```text
@图片1 中的 [特征描述] 定义为 [主体1]
@视频1 中的 [运镜/动作] 作为 [参考维度]
@音频1 中的 [音色] 作为 [背景音乐/台词音色]
```

---

## 质量检查

- [ ] 主体已正确定义并持续指代
- [ ] 每个镜头的运镜指令不超过一个
- [ ] 动作描述有肢体细化和程度量化
- [ ] 风格锚定词与 style-dictionary 匹配
- [ ] 负面约束已根据场景类型选择
- [ ] 超过 15 秒的已做多段拼接计划
- [ ] 台词用引号包裹并与画面描述分离
