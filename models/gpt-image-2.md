# GPT Image 2 — 图片引擎使用规范

> **模型**: GPT Image 2
> **定位**: 本体系中所有静态图片生成的唯一引擎
> **框架**: SCULPT（Subject · Composition · Universe · Light · Print · Texture）

---

## 一、三种生成模式

| 模式 | 英文 | 适用场景 | 特点 |
|------|------|----------|------|
| **高精度** | High | 产品图、海报、UI、精准出图 | 严格遵循提示词，文字精准，构图可控 |
| **自动** | Auto | — | 模型自行判断，**不推荐使用** |
| **创意** | Low | 插画、概念图、灵感探索 | 自由发挥，惊喜感强但不可控 |

> 规则：90% 的场景用 High 模式。只有需要 AI 自由发挥灵感时才用 Low。Auto 结果最不稳定。

---

## 二、SCULPT 七要素框架

### S — Subject 主体
**画什么？人、物、场景。越具体越好。**
```
❌ "一个手机"
✅ "一台白色iPhone 16 Pro，屏幕显示天气app，45度角"
```

### C — Composition 构图
**怎么摆放？视角、比例、位置。**
```
❌ "放在中间"
✅ "居中俯拍，产品占画面60%，周围留白，3:4竖版"
```

### U — Universe 风格
**什么调性？写实、插画、3D、赛博朋克。**
```
❌ "好看的"
✅ "极简扁平设计，柔和阴影，日式简约风格，温暖色调"
```

### L — Light 光影
**光源方向、色温、氛围感。**
```
❌ "亮一点"
✅ "左侧柔光箱打光，温暖色温3200K，浅景深f/2.0，背景虚化"
```

### P — Print 文字
**图上需要什么字？用引号包裹。**
```
❌ "写个标题"
✅ "顶部居中大字'限时5折'，白色粗体，红色圆底"
```

### T — Texture 材质/细节
**表面质感、渲染参数、后期风格。**
```
❌ "高清"
✅ "8K渲染，景深f/1.8，RAW照片质感，轻微胶片颗粒"
```

## 三、标准 Prompt 模板

```text
[Subject]，[Composition]，[Universe]风格，[Light]，
文字：[Print]，[Texture]，High mode，
no watermark，no deformation，no extra limbs，no random text
```

## 四、图片类型到比例的映射

| 类型 | 比例 | 推荐尺寸 |
|------|------|----------|
| 小红书封面 | 3:4 | 1080×1440 |
| 小红书图文 | 1:1 | 1080×1080 |
| 朋友圈 | 1:1 或 3:4 | 1080×1080 |
| 视频号封面 | 9:16 | 1080×1920 |
| 公众号封面 | 2.35:1 | 900×383 |
| Logo/Favicon | 1:1 | 512×512 |
| 电商详情页主图 | 3:4 | 750×1000 |
| 竖版海报 | 9:16 | 1080×1920 |
| 手机壁纸 | 9:16 | 1125×2436 |

## 五、编辑模式

### 核心原则：变量与不变量分离

```text
变量：告诉模型这一轮只能改什么
不变量：列出必须保持不变的所有内容
关键句："只改XXX，不要改变任何其他元素。"
```

### 编辑模板
```text
修改这张图：保持 [不变量1]、[不变量2]、[不变量3] 不变，
只将 [变量] 修改为 [新内容]。
不要改变任何其他元素。
```

## 六、多轮迭代

GPT Image 2 支持多轮对话式出图：

```
第1轮：「生成一张现代风格办公桌」
第2轮：「把桌面颜色改为深胡桃木色」
第3轮：「在桌上加一台笔记本电脑和一杯咖啡」
```

模型保留上下文，做出增量修改。

---

## 七、风格一致性

在同一系列中，将风格段落固定为 prompt 尾部不变部分：

```text
# 固定风格片段（每张复用）
High mode，日式简约风格，暖色调，浅景深f/2.0，
柔光箱打光，8K渲染，no watermark，no deformation

# 变化部分（每张修改）
[Subject]，[Composition]，文字：[Print]
```

## 八、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 文字不准 | 未用High模式或未加引号 | High模式 + 引号包裹 + 指定位置 |
| 风格漂移 | 每次prompt不同 | 固定风格片段 |
| 手指变形 | 默认模式 | High模式 + no extra limbs |
| 构图偏差 | 指令模糊 | 指定主体占比和精确位置 |
