# 极简大厂技术封面

## 视觉定位
Modern enterprise tech aesthetic, Apple / Linear / Vercel / Notion / OpenAI 风格。
极简、高级、克制。像顶级科技公司 Keynote 首页或官网 Hero。

## 画幅：16:9 横版

## 构图：左右结构
- 左侧 55%：主标题 + 副标题 + 顶部小标签
- 右侧 45%：半透明 UI 卡片 / 数据面板 / 模糊截图 / 抽象科技线条
- 50% 以上留白

## 背景
纯白 #FFFFFF / 雾灰 #F7F8FA / 极浅蓝灰 #F2F5FA
加柔和蓝紫渐变、弥散光、窗影、微弱玻璃质感
**禁止：黑色赛博朋克、霓虹、粒子、满屏代码、游戏 UI**

## 字体
中文标题：现代无衬线粗体，大号，最多 2-4 行
关键词高亮：蓝色 #4C9AFF / 蓝紫渐变，**不超过一个强调色**
副标题：浅灰色，小字号
标签：顶部小字，`·` 分隔

## UI 元素（右侧）
半透明白色 Glass Card：12-20px 圆角，轻微阴影，极细边框
内容像真实 SaaS 后台：Agent 状态 / 数据模块 / Workflow / Code 状态 / 通知
**不像游戏 HUD**

## 装饰
极细线条、渐变轨迹、小圆点节点、微弱网格、光线流动
低存在感，服务于空间

## 信息层级
1. 顶部标签（如 `真实实战 · Agent 工程`）
2. 核心标题（关键词蓝字）
3. 副标题解释
4. 底部英文关键词（`·` 分隔）

## 禁止
- 满版文字
- 大量图标
- 机器人形象
- 赛博朋克
- 夸张光效
- 电商海报感
- 培训课程感

## 输出要求
- 保留安全边距
- 主标题不贴近边缘
- 移动端缩略图可读
- 整体高级、简洁、专业
