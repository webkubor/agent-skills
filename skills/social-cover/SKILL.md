# 社交媒体封面

## 功能
生成社交媒体封面图，支持小红书、朋友圈等平台。

## 使用方式
```
把下方 Prompt 模板（连同参考图）发给任意出图模型：
GPT-Image / 即梦 / Midjourney / Seedream 均可。
```

## Prompt 模板
```
社交媒体封面（小红书风格）：产品自然融入生活场景。
柔和自然光、ins风滤镜、3:4竖版。
温暖色调，留白充足，适合加文字标题。
禁止：商业感过强、棚拍风格、过度修图。
```
