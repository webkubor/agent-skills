# 产品细节图

## 功能
输入产品图 → 输出微距特写/材质细节效果图。

## 使用方式
```
把下方 Prompt 模板（连同参考图）发给任意出图模型：
GPT-Image / 即梦 / Midjourney / Seedream 均可。
```

## Prompt 模板
```
产品微距摄影：聚焦产品关键细节（材质纹理/logo/局部功能）。
浅景深(f/1.8)，柔和自然光，高分辨率。1:1 正方形构图。
禁止：过度锐化、文字叠加、非产品元素。
```
