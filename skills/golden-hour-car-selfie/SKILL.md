# 黄金时刻车内自拍

> 风格核心：真实手机自拍 × 暖金阳光 × 无修图原生感。场景自由替换。

## 固定风格核心

```
人物类型：漂亮年轻东方女性，精致鹅蛋脸，柔和但有辨识度，非网红模板脸
肌肤标准：白皙半透明，细毛孔，淡桃色绒毛，轻微不均匀感，鼻梁和面颊自然高光，
          无重度磨皮，无塑料皮肤
画面质感：真实智能手机HDR，移动端锐度，轻微数码噪点，轻微曝光压缩，
          前置摄像头特有的轻微柔化，自然广角透视，不完美裁切，
          未经修图的手机照片外观，非单反人像，非棚拍
光线风格：暖金阳光透过车窗斜射，部分脸部保留柔和冷调阴影，
          暖金与深蓝灰车内形成自然冷暖对比，光线不完美不均匀
妆容标准：自然桃色妆，眼睑轻微暖调，极细眼线，分明睫毛，淡腮红，柔光唇
禁止元素：单反感，棚拍感，人工轮廓光，夸张浅景深，电影级调色，
          重度美颜，网红脸，娃娃脸，二次元，过度磨皮，水印，文字，Logo
```

## 可变参数

| 参数 | 默认值 | 可替换为 |
|------|--------|---------|
| 场景 | 车内（深色皮座椅/阴影内饰/窗外冷蓝日光） | 卧室窗边、咖啡厅角落、阳台、天台 |
| 服装 | 白色挂脖吊带（露肩锁骨） | 颜色/款式可换，保持简约低调 |
| 发型 | 黑长发低扎松散马尾，碎发自然垂落 | 散发/低马尾/编发，保持自然凌乱感 |
| 动作 | 手背轻托下巴，眼看镜头外 | 靠窗发呆/整理头发/看手机/闭眼听歌 |
| 构图 | 胸以上中近景，手机略高于眼平，从前右侧轻微斜拍 | 构图可调 |
| 光线色调 | 黄金时刻暖金阳光 + 冷调阴影 | 阴天柔光/室内暖灯/黄昏蓝调 |
