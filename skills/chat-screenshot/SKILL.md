# 微信聊天截图

## 功能
生成以假乱真的微信聊天截图，用于营销/信任背书。

## ⚠️ 红线
- 必须使用用户提供的头像（不自行猜测）
- 纯微信聊天界面：无拼贴、无抖音/小红书UI元素
- 出图后先私聊确认，再推群

## 使用方式
```
把下方 Prompt 模板（连同参考图）发给任意出图模型：
GPT-Image / 即梦 / Midjourney / Seedream 均可。
```

## Prompt 模板
```
微信聊天截图：纯微信聊天界面。
白色聊天气泡+绿色聊天气泡交替，包含文字对话+红包消息。
头像统一，无其他app UI元素。9:16竖版手机截图比例。
禁止：其他社交媒体界面元素、拼贴、装饰边框。
```
