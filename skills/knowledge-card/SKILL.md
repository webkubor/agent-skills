# 科普图鉴

## 功能
生成知识科普图鉴卡片，适合品种介绍、科普教育等。

## 使用方式
```
把下方 Prompt 模板（连同参考图）发给任意出图模型：
GPT-Image / 即梦 / Midjourney / Seedream 均可。
```

## Prompt 模板
```
科学插画风格的科普图鉴卡片。
主体对象居中大幅展示，科学标注线+简短文字说明。
复古自然图鉴配色，3:4竖版。
禁止：卡通化、过度装饰、与主题无关的元素。
```
