# 角色设定集（Character Model Card）

> 类型：AI 图像生成 Prompt 模板 · 引擎：gpt-image-2

## 功能

生成高级时尚品牌 Lookbook 风格的角色设定集/模特资料卡。左侧全身主视觉 + 左上角色信息 + 右上四视图 TURNAROUND + 右下六表情 EXPRESSIONS。超清商业级真实摄影质感。

## 输出要求（studio 运行时）

⚠️ 本技能的最终提示词**必须为中文**（版式内含中文文字渲染），忽略任何"转换为英文提示词"的通用要求。把用户输入映射进下方参数表，填充 Prompt 模板后**只输出填充完成的完整中文提示词本身**；用户未提供的参数按「使用示例」风格自动补全。

## 硬约束

| # | 规则 |
|---|------|
| 1 | **真实摄影** — 不是插画、不是二次元、不是 3D 渲染 |
| 2 | **同脸一致性** — 所有视图和表情必须是同一个人 |
| 3 | **服装一致** — 所有视图同一套衣服、发型、饰品 |
| 4 | **全身不裁切** — TURNAROUND 四视图中人物必须完整入镜、不裁脚 |
| 5 | **中文文字准确** — 名字和信息必须清晰可读、无乱码 |
| 6 | **英文拼写正确** — 英文名全大写、无拼写错误 |
| 7 | **8K 细节** — 皮肤纹理、服装材质、发丝层次 |
| 8 | **不要过度暴露** — 甜辣/性感≠裸露，重在搭配和气质 |

## 禁止项

❌ 二次元、漫画、塑料皮、网红脸、幼态脸
❌ 过度磨皮、夸张胸腰比例
❌ 廉价网红风、夜店感、赛博朋克
❌ 服装结构混乱、不同视图换脸
❌ 手脚畸形、多余肢体
❌ 文字乱码、英文错误
❌ 人物裁切、背景复杂
❌ 插画感、3D 渲染感
