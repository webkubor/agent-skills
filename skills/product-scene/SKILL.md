# 商品场景图

## 功能
输入一张产品照片 → 输出该产品在专业商业场景中的效果图。

## 默认参考图
[`ref-default.jpg`](https://img.webkubor.online/skills/product-scene/preview/ref-default.jpg) — 木质桌面、侧光、浅景深，产品居中陈列

## 使用方式
```
把下方 Prompt 模板（连同参考图）发给任意出图模型：
GPT-Image / 即梦 / Midjourney / Seedream 均可。
```

## Prompt 模板
```
产品商业场景摄影：将参考图中的产品替换为输入的产品图。
木质桌面，自然侧光，浅景深(f/2.8)，产品居中陈列。
暖色调、高质感、电商详情页风格。3:4 竖版构图。
禁止：卡通风格、文字水印、过曝、杂乱背景。
```
