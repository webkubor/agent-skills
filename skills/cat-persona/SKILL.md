# 猫咪拟人

## 功能
生成真实猫咪+拟人表情的创意图片。

## 默认参考图
`[`ref-default.png`](https://img.webkubor.online/skills/cat-persona/preview/ref-default.png)` — 真实猫脸+拟人表情示例

## 使用方式
```
把下方 Prompt 模板（连同参考图）发给任意出图模型：
GPT-Image / 即梦 / Midjourney / Seedream 均可。
```

## Prompt 模板
```
真实猫咪照片+拟人表情：保留猫咪真实毛发纹理和面部结构，
叠加人类表情（得意/惊讶/嫌弃/开心）。
3:4竖版，海报设计语言。
禁止：卡通猫、二次元猫、完全脱离真实猫脸。
```
