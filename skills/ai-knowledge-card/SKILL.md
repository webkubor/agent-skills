# AI 知识卡片

> webkubor 小红书「AI 百科卡片」专栏。每期一个 AI 概念，长期品牌资产。

## 固定品牌元素（每期不变）

| 元素 | 规范 |
|------|------|
| **栏目名** | `AI 知识卡片`（顶部左上角，白色小字） |
| **VOL 编号** | `VOL.01`、`VOL.02`...（顶部右上角，白色小字） |
| **页码** | `01` `02` `03` `04` `05` `06`（右下角或右上角，大号半透明数字） |
| **页脚** | `AI KNOWLEDGE CARD`（底部居中或左下，小字） |
| **栏目 Logo** | 暂不设——保持极简，只靠排版辨识 |
| **配色** | 深蓝紫渐变底 + 白色文字 + 紫色点缀 |

## 配色

```
背景渐变:     #0a0a1a → #1a0a2e （深蓝→深紫）
强调色:       #7C3AED / #8B5CF6 （紫色）
辅助色:       #4C9AFF （蓝色点缀）
文字:         #FFFFFF （白色）
弱文字:       rgba(255,255,255,0.6) （半透白）
```

## 字体

- 标题：无衬线粗体，大号
- 正文：现代无衬线，中文优先

## 结构（6 张）

| 张 | 内容 |
|----|------|
| 01 | **封面**：知识点大标题 + AI 图标 + 一句话定义 |
| 02 | **拆解**：4 个模块卡片（核心能力/要素） |
| 03 | **举例**：普通 AI vs Agent 对比流程 |
| 04 | **深入**：为什么重要 / 意义 |
| 05 | **总结**：一句话金句引用 |
| 06 | **互动**：延伸思考 + 关注引导 |

## 装饰

- 微弱粒子光点
- 模糊科技网格/轨迹
- 玻璃拟态卡片（frosted glass）
- 圆角方块模块
- 禁止：霓虹、赛博朋克、动漫、游戏 HUD

## 禁止

- 赛博朋克霓虹
- 复杂粒子爆炸
- 游戏 UI / HUD
- 动漫风格
- 文字堆满不留白
