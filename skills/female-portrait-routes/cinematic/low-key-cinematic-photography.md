## 1. 风格定位

低调电影感摄影用于生成同时具备：

```text
低调光线
+
暗背景或低照度空间
+
局部照亮
+
高对比但可读的阴影
+
低饱和电影色彩
+
真实镜头与材质
+
细腻胶片颗粒
+
安静、克制、带叙事余韵的电影静帧感
```

的写实摄影画面。

本 route 的核心不是简单堆叠：

```text
暗背景
+
电影感
+
胶片颗粒
+
高对比
```

而是建立一套明确的视觉层级：

```text
主体视觉优先
+
局部照亮
+
暗部有层次
+
材质真实
+
场景有空间纵深
+
色彩克制
+
镜头语言明确
+
画面像电影中的一帧，而不是普通暗照片
```

本 route 的“低调电影感”定义为：

```text
主体处于低照度或深暗环境中；
光线只照亮需要被看见的区域；
暗部保留层次和材质，不黑死；
画面色彩低饱和但不灰脏；
高光柔和，反光受控；
镜头具有真实焦段、景深与空间透视；
情绪安静、克制、有故事；
适用于人物、静物、产品、器物、花卉、食物和空间；
不强制近景，不强制人像，不强制直视镜头。
```

核心审美公式：

```text
暗，但不能黑死；
电影，但不能只靠黑边和颗粒；
低饱和，但不能灰蒙脏；
高对比，但不能吞没主体；
高级，但不能依赖豪华场景；
安静，但不能没有事件；
真实，但不能低清或随意曝光。
```

## 4. 路由边界

### 4.1 与黑珍珠墨金 CCD 区别

`黑珍珠墨金CCD` 更偏：

```text
柔和直闪；
人物前景明显提亮；
黑珍珠暗部；
暗金局部反光；
千禧年数码卡片机夜拍；
生活抓拍；
轻微过曝与数码噪点。
```

`低调电影感摄影` 更偏：

```text
连续光、环境光或局部造型光；
不强制闪光灯；
电影静帧；
暗部与光线方向更讲究；
低饱和电影分级；
适用于人物、静物、产品和空间。
```

如果用户强调：

```text
柔和直闪、CCD夜拍、奶油高光、黑珍珠暗部
```

优先调用：

```text
skill/routes/curve/black-pearl-dark-gold-ccd.md
```

如果强调：

```text
暗背景、局部侧光、无闪光或电影连续光、静帧叙事
```

调用本 route。

---

### 4.2 与普通暗调人像区别

`普通暗调人像` 更偏：

```text
背景暗；
人物亮；
色调偏暗；
不一定有明确电影叙事。
```

`低调电影感摄影` 必须强化：

```text
明确光源；
空间纵深；
主体状态；
材质可读；
叙事事件；
镜头焦段；
电影色彩与颗粒；
暗部层次；
高光控制。
```

---

### 4.3 与纯欲曲线生活照区别

`纯欲曲线生活照` 更偏：

```text
清纯脸 × 成年女性自然吸引力身形；
冷白生活照；
窗边、床边、靠墙等生活场景；
身体比例与曲线是重要目标。
```

`低调电影感摄影` 更偏：

```text
光线、空间、材质和情绪叙事；
人物曲线可以存在，但不是必然主目标；
也可以拍全身远景、静物、产品和空房间。
```

如果低调电影感是唯一主 Route，同时用户强调曲线时（不得加载第二条主 Route）：

```text
保留低调电影光和空间叙事；
人物服装与姿态可以增强曲线；
但面部补光、五官好看与身体结构稳定必须优先。
```

---

### 4.4 与超近景真实人脸人像区别

`超近景真实人脸人像` 更偏：

```text
面部占画面主要面积；
真实皮肤纹理；
毛孔、绒毛、唇纹与微瑕疵；
85mm 或 100mm 近景；
未修图原片。
```

`低调电影感摄影` 更偏：

```text
构图距离不限；
可近景、中景、远景；
可人物、静物、产品或空间；
重点是低调光、暗背景、局部照亮和电影叙事。
```

如果用户同时强调超近景和真实皮肤，先判断唯一主目标：

```text
面部超近景、未修图原片与皮肤微纹理是首要目标：
切换到 skill/routes/realism/ultra-close-real-face.md，不再加载本 route。

暗背景、局部连续光、空间纵深和电影静帧叙事是首要目标：
保留本 route，把真实毛孔、绒毛和唇纹转译为兼容的皮肤参数，不读取第二条主 Route。
```

---

### 4.5 与商业产品摄影区别

`商业产品摄影` 更偏：

```text
产品准确还原；
品牌识别；
材质、结构、标签和包装；
广告传播；
商品为唯一主角。
```

`低调电影感摄影` 更偏：

```text
电影光线；
暗调空间；
材质与情绪；
可以有产品，但不自动承担品牌广告任务。
```

如果用户要求：

```text
包装文字准确；
Logo；
电商主图；
广告语；
产品尺寸和颜色严格还原；
商业 KV。
```

上传服装、试衣或电商服装主图是首要目标时，切换到 `skill/routes/commercial/ecommerce-tryon.md`，不再加载本 route。
普通产品或静物仍可使用本 route 提供低调电影感摄影氛围；若上传授权产品参考图，必须同时执行产品视觉锁定，但不得承诺 Logo 与小字逐字准确。

---

### 4.6 与恐怖、悬疑风区别

本 route 可以有：

```text
神秘；
不安；
压迫；
孤独；
空旷；
夜晚；
阴影。
```

但默认不是：

```text
恐怖片；
血腥；
鬼怪；
惊悚；
暴力；
超自然实体。
```

只有用户明确要求悬疑或恐怖时，才把合规的悬念、压迫或不安转译为本 route 内部的氛围参数。注册表没有对应已实现 Route 或 Overlay 时，不得虚构文件或加载占位模块。

---

### 4.7 与普通胶片摄影区别

`普通胶片摄影` 更偏：

```text
胶片色彩；
颗粒；
柔和对比；
复古感；
可明亮或日常。
```

`低调电影感摄影` 更偏：

```text
低照度；
暗背景；
局部照亮；
空间与情绪；
电影静帧；
颗粒只作为质感辅助。
```

本 route 不允许用大量颗粒代替真实光线和电影构图。

## 7. 参数锁定规则

### 7.1 主体锁定

用户写：

```text
主体内容：一本摊开的旧书
```

必须保留。

允许扩写：

```text
一本边缘磨损、纸张泛黄的旧书摊开放在深色木桌上，书页局部被窄束暖光照亮，周围暗部保留木纹、灰尘和纸张层次。
```

不得改为：

```text
人物读书；
古董相机；
咖啡杯；
与用户输入无关的其它主角。
```

---

```text
主体状态：人物独自站在昏暗走廊中
```

必须保留。

允许扩写：

## 9. 低调光线系统

### 9.1 单一主光源原则

每张画面优先只有一个主要光源：

```text
窗边光；
门缝光；
台灯；
顶灯；
壁灯；
烛光；
城市窗外反光；
走廊尽头光；
窄束聚光；
屏幕弱光。
```

允许增加：

```text
极弱面部补光；
极弱轮廓光；
环境反射光。
```

但不得出现多个互相冲突的主光源。

---

### 9.2 局部照亮原则

局部照亮必须服务于：

```text
人物脸部；
手部动作；
物品关键结构；
产品 Logo 或材质；
书页；
花朵；
餐具；
空间尽头；
叙事道具。
```

不能随机照亮无关墙面或背景装饰。

---

### 9.3 暗部层次

暗部至少包含两到三个层级：

```text
最深暗部；
可读的深灰或深棕；
主体附近的中暗部；
局部高光。
```

避免：

```text
背景纯黑；
黑色服装与背景粘连；
空间边界消失；
材质全部变成黑色块。
```

---

### 9.4 人物面部独立补光

人物类任务必须执行：

```text
主体脸部亮度优先；
眼睛保留细小自然眼神光；
眼窝、鼻侧和嘴角阴影可控；
面部补光柔和；
补光不能改变主光方向；
补光不能形成影棚感；
皮肤纹理仍然可见。
```

推荐写法：

```text
主光负责塑造空间和侧面轮廓，极弱柔和补光从镜头侧前方轻轻抬起面部暗部，只让眼睛、鼻梁、唇形和皮肤纹理清楚，不消除原有阴影。
```

---

### 9.5 眼神光规则

眼神光必须：

```text
细小；
自然；
数量少；
位置符合主光与补光；
不过度圆亮；
不形成美容环形灯效果。
```

---

### 9.6 轮廓光规则

轮廓光可以：

```text
分离头发和背景；
勾勒肩线；
强调物体边缘；
建立空间纵深。
```

轮廓光不能：

```text
变成霓虹边缘；
亮度高过面部；
形成整圈发光；
破坏真实光源逻辑。
```

## 15. 导演式扩写规则

本 route 必须先建立一个真实、可执行、具有叙事意味的瞬间。

不得机械输出：

```text
低调光，暗背景，电影感，颗粒，人物。
```

应扩写为：

```text
夜晚旧书店即将关门，深木书架之间只剩一盏低亮度台灯和窄窗透入的微弱冷光。人物站在狭窄书架前，刚从高处抽出一本旧书，身体轻微侧向镜头，低头查看封面。台灯的暖光只照亮书页、手指和脸侧，深色书架保持层次，镜头侧前方增加极弱柔和补光，让她的眼睛、鼻梁、唇形和皮肤纹理清楚，同时不破坏低调氛围。她穿深灰哑光针织上衣，服装高光克制，黑色长发边缘有轻微轮廓光。画面使用 50mm 镜头和中等浅景深，保留书店纵深、漂浮灰尘和旧纸张质感，整体低饱和、暖棕暗调、细腻胶片颗粒，像电影中安静寻找旧书的一帧。
```

导演式扩写必须回答：

```text
主体是什么；
主体正在发生什么；
光源在哪里；
为什么只照亮这些区域；
暗部如何保持层次；
```

## 24. 质检规则

生成低调电影感摄影提示词前，必须检查：

```text
1. 主体内容是否明确？

10. 背景是否深暗但没有黑死？
11. 主体是否与背景清楚分离？
12. 色调是否低饱和但没有灰蒙脏？
13. 局部颜色是否来自真实材质、服装或灯光？
14. 是否避免全图偏蓝、偏黄、偏绿、偏红？

18. 景深是否符合叙事，而不是所有画面都过度虚化？
19. 材质纹理是否真实可读？
20. 画面是否具有空间纵深和负空间？
21. 如果主体是人物，是否明确成年？

24. 眼睛是否有细小自然眼神光？
25. 面部是否没有被顶光、侧后光或阴影吞没？

27. 皮肤是否保留真实纹理，没有塑料磨皮？
```

## 27. 最终原则

低调电影感摄影 route 的最终原则是：

```text
暗不等于黑死；
低饱和不等于灰脏；
电影感不等于颗粒和黑边；
高对比不等于吞没主体；
高级不等于豪华元素；
安静不等于没有事件；
局部照亮必须有光源逻辑；
人物暗调不能牺牲五官和颜值；
静物暗调不能牺牲材质和结构；
空间暗调不能牺牲纵深和尺度。

它应当是一张：
主体清楚、
事件成立、
光源明确、
暗部有层次、
材质真实、
色彩克制但不偏色、
镜头和景深合理、
人物五官或物体结构可读、
有安静叙事与电影余韵、
像真实电影中被截取下来的一帧摄影画面。
```
