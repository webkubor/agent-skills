# GPT Image 2 速查表

## 一句话口诀
> S-C-U-L-P-T：主体(Subject) · 构图(Composition) · 风格(Universe) · 光影(Light) · 文字(Print) · 材质(Texture)

## 模式选择
```
日常出图 → High（默认）
创意探索 → Low
不要用 Auto
```

## 标准模板
```
[主体描述]，[构图方式]，[风格调性]，[光影设置]，[文字]，
High mode, 8K, depth of field f/1.8, RAW photo quality,
subtle film grain, no watermark, no deformation, no extra limbs
```

## 编辑模式关键句
```
只改 [具体元素]，不要改变任何其他元素。
```

## 图片比例速查
| 平台 | 比例 | 尺寸(建议) |
|------|------|-----------|
| 小红书封面 | 3:4 | 1080×1440 |
| 小红书图文 | 1:1 | 1080×1080 |
| 朋友圈 | 1:1 或 3:4 | 1080×1080 |
| 视频号封面 | 9:16 | 1080×1920 |
| 公众号封面 | 2.35:1 | 900×383 |
| Logo/Favicon | 1:1 | 512×512 |
| 电商详情页 | 3:4 | 750×1000 |
| 竖版海报 | 9:16 | 1080×1920 |

## 常见问题速解
| 症状 | 方案 |
|------|------|
| 文字不对 | 引号包裹 + 指定位置 + High模式 |
| 风格不统一 | 每次用相同风格锚定词（见style-dictionary） |
| 手指变形 | High模式 + no deformation no extra limbs |
| 主体模糊 | 增加特征描述（2-3个静态特征） |
